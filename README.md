# FSC Theater — Official Cinema Website

A fully responsive, modern cinema website for The FSC Theater,
built from scratch using HTML, CSS, and JavaScript.

Features:
  - Movie listings & showtimes
  - Interactive seat selection
  - Ticket booking interface
  - Upcoming releases section
  - Responsive design for all screen sizes

Tech Stack:
  - HTML5
  - CSS3 (Flexbox & Grid)
  - Vanilla JavaScript
